Guia para leigo completo

Como usar o app sem medo de errar.

Esta página foi feita para quem não mexe com planilha, banco de dados ou sistemas. Você só precisa abrir o painel, preencher os dados da coleta e usar os botões certos para guardar tudo com segurança.

Primeiros passos

O que esse app faz

Pense nele como um caderno inteligente: você coloca as medidas e o site monta os gráficos, o histórico e os comparativos sozinho.

1

Ele guarda as coletas

Cada vez que você registra uma nova avaliação, o site salva a data, os tempos do RAST, o salto, o peso, a altura e as observações.

2

Ele atualiza os gráficos

Depois de salvar, os gráficos e os cards mudam automaticamente para mostrar evolução, queda, comparação e leitura do momento.

3

Ele não usa servidor

Os dados ficam no navegador. Por isso é importante exportar backup em JSON ou Excel sempre que fizer mudanças importantes.

Rotina ideal

O jeito mais seguro de usar

Se seguir essa ordem, fica bem difícil perder dado ou se enrolar.

1

Abra o painel

Entre em index.html e confira se o atleta certo já aparece no histórico.

2

Cadastre a coleta

Preencha o formulário com os dados do dia e clique em Salvar coleta.

3

Revise os gráficos

Veja se os números subiram ou caíram e confira se as fotos do shape ficaram anexadas.

4

Faça backup

Ao terminar, exporte pelo menos o JSON. Assim você consegue recuperar tudo depois.

Cadastro de coleta

Como preencher cada campo

Se você souber preencher este bloco, o resto do app praticamente trabalha sozinho.

Atleta

Escolha Noah ou Marcelo. Isso define em qual histórico a coleta entra.

Data

Use a data real da avaliação. Isso é o que organiza a linha do tempo dos gráficos.

Idade, peso e altura

Coloque exatamente como estava no dia do teste. O app usa isso para calcular indicadores.

SLJ

Digite a melhor distância do salto horizontal em metros. Exemplo: 1.54.

RAST

Preencha os 6 tiros com os tempos em segundos. Exemplo: 7.16, 7.70 e assim por diante.

Observações

Escreva algo útil do dia, como cansaço, dor, retorno de pausa, mudança de treino ou qualquer detalhe que explique o resultado.

Dica importante

Nos campos com número decimal, use ponto. Exemplo: 1.54 em vez de 1,54. Isso evita erro no cálculo.

Fotos do shape

Como anexar as imagens do dia

Essas fotos ficam ligadas à coleta específica, então cada dia pode ter seu próprio registro visual.

1

Escolha as fotos

Clique no campo de fotos e selecione uma ou mais imagens do computador ou celular.

2

Veja a prévia

Antes de salvar, a prévia aparece na lateral. Isso serve para confirmar se as fotos certas foram escolhidas.

3

Salve a coleta

Depois de salvar, as fotos aparecem dentro do card daquele dia no histórico do atleta.

Backup e segurança

Como não perder os dados

Essa é a parte mais importante para quem vai usar o app por meses.

Exportar JSON

É o backup principal. Guarde esse arquivo numa pasta segura, nuvem ou WhatsApp para você mesmo.

Exportar Excel

Serve para ter os dados em planilha e consultar fora do app. Bom para mandar para outra pessoa.

Importar backup

Se trocar de navegador, computador ou perder os dados, clique em Importar backup e selecione o JSON salvo.

Regra simples

Fez uma coleta nova? Exporte um JSON novo no mesmo dia. Esse é o hábito que mais evita dor de cabeça.

Relatório rápido

Como gerar algo para mostrar ou imprimir

O próprio app já deixa isso bem mastigado.

PDF do atleta

Entre no perfil individual e clique em Imprimir PDF. O navegador abre a tela de impressão.

O que sai no PDF

Cards, gráfico, diagnóstico resumido, histórico e indicadores principais do atleta selecionado.

Comparar rápido

Na página inicial, a área de comparativo mostra quem está melhor em cada indicador mais importante.

Dúvidas comuns

Perguntas que um leigo normalmente faz

Se acontecer alguma dessas situações, a solução costuma ser simples.

Os dados sumiram. E agora?

Tente abrir no mesmo navegador em que você cadastrou. Se não voltar, use Importar backup e selecione o JSON que você exportou antes.

Posso usar em outro computador?

Pode. Mas os dados não vão junto sozinhos. Você precisa levar o arquivo de backup JSON e importar no outro computador.

Errei uma coleta. Tem como apagar?

Tem. No histórico do atleta, cada card de avaliação tem o botão Remover coleta.

Posso zerar tudo?

Sim. Existe o botão Restaurar dados iniciais. Use isso só se você tiver backup, porque apaga o que foi salvo nesse navegador.

Preciso preencher tudo sempre?

Para a coleta funcionar direito, sim. O app depende desses campos para calcular os gráficos e indicadores automaticamente.

Qual backup eu devo confiar mais?

O principal é o JSON. O Excel é ótimo para leitura, mas o JSON é o que reconstrói o app.
